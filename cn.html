COMPUTER NETWORK


>> COMPUTER NETWORKING refers to interconnecting computing devices that can exchange data and share resources with each other. These NETWORK devices use a system of rules, called communications protocols, to transmit information over physical or wireless technologies.



WORK OF COMPUTER NETWORK

>> NODES and LINKS are the basic building blocks of computer network. A network may be data communication equipment(DCE) such as a modern, hub, switch or data terminal equipment(DTE). It basically provides communication between networks physical components and functions.



ARCHITECHTURE

\

  • >> COMPUTER NETWORK falls under two broad catogories:

  • 1.CLIENT-SERVER ARCHITECHTURE

  • >>In this type of computer network, SERVER NODES provides the resources. CLIENT nodes makes communication.

  • 2.PEER-TO-PEER ARCHITECTURE

  • >>This is a type of network where a group of computers acts as a nodes consists of decentralized clients and servers.
